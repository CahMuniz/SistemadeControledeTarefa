# 📋 Controle de Tarefas

Sistema web desenvolvido para auxiliar no gerenciamento de tarefas de suporte técnico, permitindo cadastrar, editar, excluir e acompanhar atividades por categoria, prioridade e status.
**Link projeto https://cahmuniz.github.io/SistemadeControledeTarefa/**

## 🚀 Funcionalidades

- ✅ Cadastro de novas tarefas
- ✏️ Edição de tarefas
- 🗑️ Exclusão de tarefas
- 🔍 Filtro por status
- 📂 Filtro por categoria
- 🚦 Definição de prioridade
- 💾 Armazenamento das informações utilizando LocalStorage
- 📱 Interface responsiva e intuitiva

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage (armazenamento local do navegador)

## ⚙️ Como o projeto foi desenvolvido

O sistema foi desenvolvido utilizando a arquitetura de aplicações web no lado do cliente (Client-Side). A interface foi construída em **HTML5**, a estilização realizada com **CSS3** e toda a lógica da aplicação implementada em **JavaScript**.

As tarefas são armazenadas no **LocalStorage** do navegador, permitindo que os dados permaneçam salvos mesmo após fechar a página, sem a necessidade de um banco de dados.

O sistema permite:

- Criar tarefas;
- Editar informações;
- Excluir tarefas;
- Filtrar por categoria e status;
- Organizar tarefas por nível de prioridade.

## ▶️ Como executar

1. Faça o download ou clone este repositório.
2. Abra o arquivo **index.html** em qualquer navegador moderno.
3. Comece a cadastrar e gerenciar suas tarefas.
